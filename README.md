## Deskripsi
Repository ini berisi 3 aplikasi CRUD Mahasiswa sebagai bagian dari technical test 
LAMDA Technology Solution. Setiap aplikasi menggunakan stack teknologi yang berbeda 
namun memiliki fungsi yang sama, yaitu manajemen data mahasiswa dengan relasi ke data jurusan.
## Cara Melihat Tiap Aplikasi
Pindah ke branch yang sesuai dan baca README masing-masing:
```bash
git checkout soal-1   
git checkout soal-2  
git checkout soal-3
```
## Fitur Umum (semua aplikasi)
- CRUD data mahasiswa (Create, Read, Update, Delete)
- Search / filter data mahasiswa
- Relasi tabel mahasiswa dan jurusan
- Validasi input
